# v15L2 — Round tables match classroom folding banquet surface

## Motivation
User wanted the round stacking tables and the classroom folding banquets to
visually match, since they share the same use case (temporary event tables).

## Changes in `build_round_table()`
- Top material: `MAT_class_top` (0.93, 0.91, 0.84, roughness 0.38) — identical
  to the folding banquet laminate
- Edge treatment: `MAT_class_bump` (0.78, 0.75, 0.68, roughness 0.55) T-mold
  bullnose ring around the full 48" circumference (1.4" tall, 0.25" radial depth)
- Top thickness bumped 1.15" → 1.25" to match classroom exactly
- 64-vertex smooth-shaded cylinder for a clean rounded disc look
- Chrome splayed legs (`MAT_roundtbl_leg`), black plastic foot caps, X cross-brace retained
